# Actor API Reference

Complete reference for creating actors. Actors are TypeScript modules that draw on a shared Pixi.js canvas.

## File Structure

Generated by `pnpm new:actor <name>`. Only modify `src/index.ts`.

```
actors/community/<name>/
  src/index.ts       # Actor implementation (edit this)
  package.json       # Auto-generated
  tsconfig.json      # Auto-generated
  vite.config.ts     # Auto-generated
```

```typescript
import { registerActor } from '@art/actor-sdk';
import type { Actor, ActorMetadata, ActorSetupAPI, ActorUpdateAPI, FrameContext } from '@art/types';

// ... define actor ...

registerActor(actor);    // REQUIRED — registers with runtime
export default actor;    // REQUIRED
```

## Actor Interface

```typescript
interface Actor {
  metadata: ActorMetadata;
  setup?(api: ActorSetupAPI): Promise<void>;         // Called once on activation
  update(api: ActorUpdateAPI, frame: FrameContext): void;  // Called every frame (60fps)
  teardown?(): Promise<void>;                         // Called on deactivation
  onContextChange?(context: ContextAPI): void;        // Called on environment changes
}

interface ActorMetadata {
  id: string;                    // kebab-case, unique (e.g., 'spiral-bloom')
  name: string;                  // Display name
  description: string;           // What it does
  author: { name: string; github?: string };
  version: string;               // Semver (e.g., '1.0.0')
  tags: string[];                // Categories
  createdAt: Date;
  role?: 'background' | 'foreground' | 'filter';  // Default: 'foreground'
  preferredDuration?: number;    // Seconds
  requiredContexts?: Array<'time' | 'weather' | 'audio' | 'video' | 'social' | 'display'>;
}

interface FrameContext {
  deltaTime: number;   // ms since last frame
  frameCount: number;  // Frames since actor started
  time: number;        // ms since actor started (use for animation)
}
```

**Roles:** background (exactly 1 per cycle), foreground (2–5, default), filter (0–2 per layer, uses `api.filter.*` only).

### Gradients (CRITICAL — 0-1 coordinates only)

All gradient coordinates are **relative to the shape's bounding box** (0 = left/top, 0.5 = center, 1 = right/bottom). Using absolute pixel values causes `TypeError: Cannot read properties of null (reading 'style')`.

```typescript
interface Gradient {
  type: 'linear' | 'radial';
  stops: { offset: number; color: Color }[];
  // Linear: x0?, y0?, x1?, y1? (default: top-to-bottom)
  // Radial: cx?, cy?, radius? (default: centered, 0.5 radius)
}

// Centered radial glow:
{ type: 'radial', cx: 0.5, cy: 0.5, radius: 0.5, stops: [{offset:0, color:'white'}, {offset:1, color:'transparent'}] }
// Horizontal linear:
{ type: 'linear', x0: 0, y0: 0.5, x1: 1, y1: 0.5, stops: [...] }
```

Read-only access to current canvas state.


## Mandatory Constraints

| Rule | Detail |
|------|--------|
| Gradient coordinates | Must be 0–1 range (relative to shape). Pixel values crash. |
| Blend modes | Pass `blendMode` in each shape's style. Never call `setBlendMode()` in `update()`. |
| Memory: pre-allocate | Create all arrays/objects in `setup()`. Use object pools with `active` flag. Never allocate in `update()`. |
| Memory: cap collections | Define MAX constants, enforce limits. No unbounded growth. |
| Memory: no array mutation | No `.push()`, `.filter()`, `.splice()`, `.shift()` in `update()`. Use index-based iteration. |
| Alpha visibility | Use >= 0.6 for main shapes. Skip rendering when `alpha < 0.05`. |
| Stroke width | Use >= 2.5px for visible lines. |
| Numeric colors | Prefer `0xRRGGBB` with separate `alpha` param over string colors. |
| Security sandbox | No `fetch`, `WebSocket`, `localStorage`, `document.*` (except `createElement('canvas')`), `window.*`, `eval`, `import()`. |
| Draw calls | Target < 300 per frame for 60fps. |
| Filters | Max 3–5 per actor. Prefer low-cost filters (noise, vignette, pixelate). |
| Teardown | Reset all state, release references. |
| registerActor | Must call `registerActor(actor)` at end of file. |
| Dark/light mode | Check `api.context.display.isDarkMode()` and adapt colors/alpha. |

## Example: Foreground Actor

```typescript
import { registerActor } from '@art/actor-sdk';
import type { Actor, ActorMetadata, ActorSetupAPI, ActorUpdateAPI, FrameContext } from '@art/types';

const metadata: ActorMetadata = {
  id: 'pulse-dots',
  name: 'Pulse Dots',
  description: 'Drifting dots that pulse and connect with nearby neighbors',
  author: { name: 'Hackathon Dev', github: 'devname' },
  version: '1.0.0',
  tags: ['dots', 'connections', 'ambient'],
  createdAt: new Date(),
  preferredDuration: 30,
  requiredContexts: ['display'],
};

const MAX_DOTS = 12;
interface Dot { active: boolean; x: number; y: number; vx: number; vy: number; phase: number }
let dots: Dot[] = [];
let canvasW = 0;
let canvasH = 0;

const actor: Actor = {
  metadata,

  async setup(api: ActorSetupAPI) {
    const size = api.canvas.getSize();
    canvasW = size.width;
    canvasH = size.height;
    dots = [];
    for (let i = 0; i < MAX_DOTS; i++) {
      dots.push({
        active: true,
        x: Math.random() * canvasW,
        y: Math.random() * canvasH,
        vx: (Math.random() - 0.5) * 0.8,
        vy: (Math.random() - 0.5) * 0.8,
        phase: Math.random() * Math.PI * 2,
      });
    }
  },

  update(api: ActorUpdateAPI, frame: FrameContext) {
    const t = frame.time / 1000;
    const isDark = api.context.display.isDarkMode();
    const dotColor = isDark ? 0x88ccff : 0x2266aa;
    const lineColor = isDark ? 0x4488cc : 0x3366aa;

    for (let i = 0; i < dots.length; i++) {
      const d = dots[i];
      if (!d.active) continue;

      // Move and bounce
      d.x += d.vx * frame.deltaTime * 0.06;
      d.y += d.vy * frame.deltaTime * 0.06;
      if (d.x < 0 || d.x > canvasW) d.vx *= -1;
      if (d.y < 0 || d.y > canvasH) d.vy *= -1;
      d.x = Math.max(0, Math.min(canvasW, d.x));
      d.y = Math.max(0, Math.min(canvasH, d.y));

      // Pulsing radius
      const r = 8 + Math.sin(t * 2 + d.phase) * 4;
      api.brush.circle(d.x, d.y, r, { fill: dotColor, alpha: 0.7, blendMode: 'add' });

      // Connect nearby dots
      for (let j = i + 1; j < dots.length; j++) {
        const o = dots[j];
        if (!o.active) continue;
        const dx = o.x - d.x, dy = o.y - d.y;
        const distSq = dx * dx + dy * dy;
        if (distSq > 22500) continue; // 150px threshold
        const alpha = 0.5 * (1 - Math.sqrt(distSq) / 150);
        if (alpha < 0.05) continue;
        api.brush.line(d.x, d.y, o.x, o.y, { color: lineColor, width: 1.5, alpha });
      }
    }
  },

  async teardown() {
    dots = [];
    canvasW = 0;
    canvasH = 0;
  },
};

registerActor(actor);
export default actor;
```

## Example: Filter Actor

```typescript
import { registerActor } from '@art/actor-sdk';
import type { Actor, ActorMetadata, ActorUpdateAPI, FrameContext } from '@art/types';

const metadata: ActorMetadata = {
  id: 'glitch-shift',
  name: 'Glitch Shift',
  description: 'Pulsing chromatic aberration with subtle noise',
  author: { name: 'Hackathon Dev', github: 'devname' },
  version: '1.0.0',
  tags: ['filter', 'glitch', 'chromatic'],
  createdAt: new Date(),
  role: 'filter',
};

let speed = 0.5;
let intensity = 4;

const actor: Actor = {
  metadata,

  async setup() {
    speed = 0.3 + Math.random() * 0.4;
    intensity = 3 + Math.random() * 4;
  },

  update(api: ActorUpdateAPI, frame: FrameContext) {
    const t = frame.time / 1000;
    const pulse = Math.sin(t * Math.PI * 2 * speed);
    const offset = intensity * (0.5 + pulse * 0.5);
    const angle = t * 0.3;

    api.filter.chromaticAberration(
      [Math.cos(angle) * offset, Math.sin(angle) * offset],
      [Math.cos(angle + Math.PI) * offset, Math.sin(angle + Math.PI) * offset]
    );
    api.filter.noise(0.04 + pulse * 0.03);
    api.filter.vignette(api.context.display.isDarkMode() ? 0.15 : 0.08, 0.7);
  },

  async teardown() {
    speed = 0.5;
    intensity = 4;
  },
};

registerActor(actor);
export default actor;
```

## Common Pitfalls

| Symptom | Cause | Fix |
|---------|-------|-----|
| `TypeError: null (reading 'style')` | Gradient with pixel coordinates | Use 0–1 range for all gradient coords |
| Shapes invisible | Low alpha or dark colors on dark bg | Alpha >= 0.6, use `isDarkMode()` to adapt |
| Entire canvas flickers | `setBlendMode()` called in update loop | Pass `blendMode` in each shape's style object |
| Objects drift off-screen | Transform state not restored | Always pair `pushMatrix()` / `popMatrix()` |
| Memory leak / frame drops | Allocating arrays/objects in `update()` | Pre-allocate in `setup()`, use object pools |
| Validation fails: security | Using fetch, localStorage, document, etc. | Actors run in a sandbox — no network/storage/DOM |
| Actor not appearing | Missing `registerActor()` call | Add `registerActor(actor)` before `export default` |
| Actor not in dev server | Dev server not restarted after creation | Restart `pnpm dev` (Vite glob evaluated at startup) |
